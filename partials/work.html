Bowling Behind the Scenes

This was my final project during my time at Galvanize. The goal of the project was to teach the user how the scoring system works in bowling. There is also a scoreboard feature built with socket.io. The point of this is to allow several users to keep track of their scores even if they are not bowling in the same place. Checkout the deployed version here and the code here.

Tea Shopping

This is a mock application for a site that sells various kinds of tea. The user is able to select different teas and quantities and add them to a shopping cart. Then the user can proceed to a checkout screen and see their total. This app uses Angular.js and Bootstrap. Deployed site here and code here.

Carefree Dental

This is a mock dental savings calculator. The user can select the appropriate amount of family members and a table is generated. The user can then select various services and see an updating total that shows how much they would save by switching to Carefree Dental. The savings data is currently fake but it is developed in a way that would allow real data to be implemented easily. Deployed here, code here.
